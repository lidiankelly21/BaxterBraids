# 📖 Descrição

Projeto em desenvolvimento com o objetivo de criar uma Landing Page moderna, responsiva e intuitiva para a divulgação de serviços de tranças. A aplicação foi projetada para valorizar o trabalho da profissional, apresentar seus principais serviços e facilitar o contato com potenciais clientes.

A página contará com informações essenciais, como apresentação da trancista, catálogo de estilos de tranças, portfólio de trabalhos realizados, tabela de preços e canais de atendimento. Além disso, a interface está sendo desenvolvida para proporcionar uma experiência de navegação agradável e acessível em diferentes dispositivos.

O principal objetivo do projeto é fortalecer a presença digital da profissional, aumentar a visibilidade dos seus serviços e converter visitantes em clientes por meio de uma comunicação clara e de um processo simplificado para solicitação de orçamentos e agendamentos.
